Você deve desenvolver um aplicativo móvel usando Flutter para fazer o consumo de alguma API da Internet. Você pode criar uma API própria ou consumir uma API já existente.
Seu app deve necessariamente realizar requisições GET e POST, para obter e enviar dados para a aplicação Web consumida.
O app deve ter uma interface que permita ao usuário listar as informações obtidas via requisição GET e uma tela para enviar dados para a aplicação via POST.
Você deve enviar o código fonte do seu app.
